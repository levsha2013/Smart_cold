# 🧊 Smart Holodilnik — умный холодильник

Self-hosted сервис учёта продуктов: количество, даты изготовления и сроки годности,
напоминания об истечении срока, список покупок, места хранения. Управление через
веб-интерфейс, уведомления — в Telegram. Заложен AI-слой для добавления продуктов
по фото и голосу.

## Возможности

- Продукты: название, категория, количество + единица, дата изготовления, срок годности, место хранения
- Статус свежести: 🟢 свежий / 🟡 скоро истечёт / 🔴 истёк (учитывает вскрытие)
- Словарь типовых сроков (`data/shelf_life.json`): для яиц, молока и т.п. срок предлагается автоматически
- Операции: добавить, списать, отметить вскрытым, удалить — всё пишется в историю
- Список покупок (автодобавление при списании продукта в ноль)
- Telegram-дайджест об истекающих продуктах (ежедневно)
- Добавление через **текст / фото / голос** (фото — OpenRouter vision, голос — Groq Whisper)
- Рецепты — заглушка под будущую LLM-интеграцию

## Запуск (Docker)

```bash
cp .env.example .env   # заполните токены при необходимости
docker compose up --build
```

Откройте http://localhost:8000

## Запуск (локально)

```bash
python -m venv .venv
.venv/Scripts/activate            # Windows;  source .venv/bin/activate — Linux/macOS
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## Конфигурация (`.env`)

| Переменная | Назначение |
|---|---|
| `WARN_DAYS` | За сколько дней до срока продукт считается «скоро истечёт» |
| `DIGEST_HOUR` | Час ежедневного дайджеста (UTC) |
| `TELEGRAM_TOKEN`, `TELEGRAM_CHAT_ID` | Уведомления в Telegram (пусто → пишется в лог) |
| `OPENROUTER_API_KEY` | Распознавание продуктов по фото (пусто → заглушка) |
| `GROQ_API_KEY` | Распознавание речи (пусто → заглушка) |

Без ключей сервис полностью работает — AI-функции показывают «не настроено».
